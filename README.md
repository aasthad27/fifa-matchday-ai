# FIFA Matchday AI

FIFA Matchday AI is a modern fan and staff experience app for a football matchday event. It brings together match insights, travel planning, live updates, interactive dashboards, and a polished user experience in one place.

Live Demo: https://fifa-matchday-ai.vercel.app/

## Features

### 1. Fan Dashboard
The app offers a welcoming fan home screen with a polished layout, quick-access shortcuts, and an intuitive experience for navigating matchday activities. Fans can quickly jump to upcoming matches, transport planning, photo moments, and predictions without feeling overwhelmed.

### 2. Match Schedule and Match Details
Users can view upcoming fixtures and explore detailed match information such as venue, timing, teams, and match-related updates. The experience is structured to make event information easy to scan and understand.

### 3. Live Matchday Updates
The app provides real-time-style updates and reminders so fans stay informed about important matchday alerts, schedule changes, and traffic or event notifications.

### 4. Transport and Stadium Guidance
A dedicated transport planning section helps users navigate to the venue with useful travel information and route awareness. This makes the overall experience more practical and convenient for fans attending live events.

### 5. Photo Gallery and Media Experience
Fans can browse a visual collection of matchday moments and media content, giving the app a more immersive and engaging feel beyond just text-based information.

### 6. Predictions and Fan Engagement
The app includes prediction features and interactive stats that encourage users to participate, compare performance, and feel more connected to the event.

### 7. Staff Tools and Operations Dashboard
For staff and organizers, the platform includes tools for ticket management, crowd flow monitoring, sentiment analysis, and sustainability tracking. These help create a more organized and efficient event management experience.

### 8. Accessibility-Friendly Design
The interface is built with clarity and usability in mind. It uses readable layouts, simple navigation, and a calm visual approach to ensure the experience feels accessible and easy to use for a wide range of users.

### 9. Modern UI and Responsive Experience
The app uses smooth animations, modern visual styling, and responsive design principles to deliver a premium-quality experience on both mobile and desktop screens.

## Tech Stack

- React.js
- Vite
- React Router DOM
- Framer Motion
- Lucide React
- Tailwind CSS
- Recharts

## Project Structure

- src/components/fan/ - Fan experience screens
- src/components/staff/ - Staff experience screens
- src/context/ - Global app context
- src/data/ - Match, prediction, transport, and stadium data
- src/layout/ - Fan and staff layout shells

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/fifa-matchday-ai.git
   cd fifa-matchday-ai
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Run locally
   ```bash
   npm run dev
   ```

4. Build for production
   ```bash
   npm run build
   ```

## Deployment

This project is deployed on Vercel:

- Open the app and choose between Fan mode and Staff mode
- Sign in using the demo credentials shown in the UI
- Explore match updates, transport planning, predictions, and staff tools
